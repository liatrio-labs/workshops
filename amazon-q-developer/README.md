# Amazon Q Developer Workshop

Build a **Parking Garage Management API** using Amazon Q Developer with test-driven development.

## Quick Start

1. Install prompts: `./install-prompts.sh`
2. Follow: `workflow.md`

## Prerequisites

- Amazon Q Developer active in IDE
- Git initialized
- Node.js or Python knowledge

## Files

- `workflow.md` - 11-step development process
- `prd.md` - API requirements
- `prompts/` - AI prompt templates
- `install-prompts.sh` - Setup script

## What You'll Learn

- AI-assisted development with Amazon Q Developer
- Test-driven development workflow
- Code review and quality practices
- Requirements to working API (15-30 minutes)

## Usage

```bash
./install-prompts.sh
# Then use prompts like:
@project-user-stories @prd.md
@dev @docs/features/01-list-spots.md
```

## Customization

Replace `prd.md` with your own requirements. Modify prompts for different tech stacks.
